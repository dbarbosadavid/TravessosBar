## Projeto em C#

### Projeto realizado para fins acadêmicos

Implementação de um Sistema para registro de Cardápio e de Pedidos que permite CRUD de:

* Usuários
* Pedidos

FRONT-END (Windows Forms)
Banco de Dados (SQL-Server)
